# scripts/update_and_check.sh

## macOS / Linux
1. `cd` vers la racine du dépôt.
2. Exécuter `./scripts/update_and_check.sh`.
3. Suivre les messages du log `scripts/logs/update_and_check.log` en cas d'échec.

## Windows (PowerShell)
1. `cd` vers la racine du dépôt.
2. Utiliser `bash ./scripts/update_and_check.sh` (via Git Bash ou WSL).
3. Consulter `scripts/logs/update_and_check.log` pour les détails.
